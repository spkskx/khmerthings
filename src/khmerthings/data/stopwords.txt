# khmerthings stopword lexicon: function words tagged by category, for the
# `condense` content-word extractor. One entry per line, two tab-separated
# columns: word<TAB>category. UTF-8, NFC, Khmer letters/marks only. Blank
# lines and '#' comments are ignored.
#
# Every word here must also appear in one of the word files
# (words/names/modern) — a stopword is a real word we choose to strip, not a
# new spelling (enforced by tests). Categories are a closed set (see
# STOPWORD_CATEGORIES in condense.py).
#
# The categories split into two intent-relevant groups:
#   - Stripped by default (low information for intent): particle, politeness,
#     filler, preposition, conjunction, demonstrative.
#   - Kept by default (they carry intent and are only removed on request):
#     pronoun, auxiliary, question.
# This list is deliberately conservative — verb/preposition-ambiguous words
# (ទៅ "go"/to, មក "come"/toward, ណា "which"/softener) are left out rather than
# risk stripping content.
